# game-icons-font

Create web font from all http://game-icons.net icons.

Preview all font icons http://game-icons-font.net/full_list/.

## Installation

This plugin requires Nodejs, npm, Grunt.

## Compile all

```
grunt default
```

## Compile Font (fonts)

```
grunt compilefont
```

## Compile Css

```
grunt compilecss
```
